# assignments-webb3-Zakariya

Detta repository innehåller mina inlämningsuppgifter i kursen Webbteknik 3 (1ME323).

## Innehåll
- `index.html` – landningssida för inlämningsuppgifter  
- `global.css` – global CSS som används på alla sidor  
- `js/main.js` – huvudscript som kör navigation och cards  
- `js/assignments.js` – datastruktur (array av objekt) med alla uppgifter (id, title, link, description)  
- `js/nav.js` – bygger global navigation dynamiskt och markerar aktuell sida  
- `js/cards.js` – bygger “kort” på landningssidan utifrån samma datastruktur  
- `js/about.js` – information om mig (namn, kurs, användarnamn)  
- `assignments1/` – sida för Uppgift 1 (egen HTML och egen CSS)

## Funktionalitet
- Navigation och kort genereras dynamiskt med JavaScript från en gemensam datastruktur  
- Den aktuella sidan markeras i navigationen via en `active`-klass
